
Reports are created with create_reports()

data must be .csv or .xls (eventually) or RData from prior fit.

figures will be saved in doc/figures/ESU_figures folder.

The doc/tex files folder are some wrappers that are used to make the tables that go into the ESU reports.  Don't delete.  They call the tex files made by create_reports.R for the tables.

The esu_report.Rmd is the file that makes the report.  Don't delete.
